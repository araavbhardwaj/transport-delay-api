# Transport Delay Analytics API

## Overview
This project is a fully functional, data-driven RESTful API designed to track, manage, and analyse public transport delays. It features secure JWT authentication, complete CRUD operations, and complex SQL data aggregation to identify traffic hotspots and temporal trends.
