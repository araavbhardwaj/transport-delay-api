# Transport Delay Analytics API

## Overview
This project is a RESTful API that allows users to report and analyse transport delays.

## Features
- CRUD operations for incidents
- Analytics endpoint for average delays
- Built with FastAPI and SQLite

## Setup Instructions

1. Install dependencies:
pip install -r requirements.txt

2. Run server:
uvicorn main:app --reload

3. Open API docs:
http://127.0.0.1:8000/docs

## Technologies Used
- FastAPI
- SQLAlchemy
- SQLite
